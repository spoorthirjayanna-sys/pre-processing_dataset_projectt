# Pre-processing Dataset Project

## Description
This project focuses on preprocessing and cleaning a dataset using Python.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Files Included
- Sales-Export_2019-2020.csv – Original dataset
- cleaned_dataset.csv – Cleaned dataset after preprocessing
- preprocessing_dataset_code.ipynb – Python notebook used for data cleaning

## Steps Performed
1. Loaded the dataset
2. Checked dataset information
3. Handled missing values
4. Performed data preprocessing
5. Saved the cleaned dataset using df.to_csv()
